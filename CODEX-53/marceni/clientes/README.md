# Clientes (dentro do CODEX)

Pasta para guardar materiais brutos recebidos/coletados de clientes (prints, textos, logos, fotos, links).

Padrao de nome (recomendado):
- `YYYY-MM-DD_origem_assunto.ext`

Exemplos:
- `2026-03-02_instagram_bio.png`
- `2026-03-02_whatsapp_dados.md`
- `2026-03-05_foto_perfil.jpg`

Dica: se estiver com pressa, jogue tudo em `CODEX-53/marceni/clientes/<cliente>/inbox/` e a gente organiza depois.
